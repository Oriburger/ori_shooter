## One Page Proposal 📜

---

### 게임 설명

- 연구실에서 탈출한 실험 쥐가 살아남기 위해 끝 없는 길을 달린다
- 각종 장애물들과 방해꾼들을 무찌르고 앞으로 나아가자!

### 게임 구성

- 목표 : 뒤에서 다가오는 독가스와 앞에서 방해하는 벌레 군단을 무찌르고 나아가자!
- 조작 : 자동 달리기, 키보드 / 마우스
- 시스템 : Endless Runner

### 게임의 핵심 재미

- 기존의 Endless Runner 게임과 TPS 장르의 결합
    - 뒤에서 다가오는 가스로 인한 긴장감
    - 앞에서 방해하는 벌레들을 해치우는 액션
- 점점 더 빨라지는 속도와 어려워지는 난이도에 몰입감 선사
